# Retention Analysis Project

This project was developed as part of my BBA (Data Science & AI), Semester 2, under the guidance of Mr. Mahadev Bag.

## Objective
To analyze user retention patterns and identify key drop-off points in user engagement.

## Dataset
The dataset consists of user retention data tracked over multiple days, focusing on how many users continue to engage after initial interaction.

## Key Insight
A significant drop-off was observed after Day 2, where retention decreased from approximately 60% to 35%. This suggests that while initial user interest is strong, maintaining engagement beyond the early stage is a challenge.
